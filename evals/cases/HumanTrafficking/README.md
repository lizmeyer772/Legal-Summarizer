# HumanTrafficking

This case folder contains one evaluation document and the files used to assess model performance.

## Contents

`source/`
Original source document for this evaluation case.

`rubric.json`
Checklist of the facts, risks, deadlines, and next steps the model should capture.

`generated-output.md`
Place to paste the actual summary and attorney next steps generated by the app.

`references/`
Optional attorney-reviewed reference outputs or notes.
