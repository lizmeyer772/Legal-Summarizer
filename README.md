# LLM-Based Legal Complaint Analyzer

Work in progress!

This project is a full-stack LLM application built to help legal professionals quickly review and act on new cases. Users can upload a legal complaint, and a LLaMA-based model will generate a clear summary along with suggested next steps and/or discovery questions for the attorney. The front end is built in React, with Python handling the LLM and document-processing pipeline. Right now the app is focused on complaint summarization, structured attorney follow-up, and evaluation workflows for legal-document review.
