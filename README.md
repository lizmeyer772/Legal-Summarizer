# LLM-Based Legal Complaint Analyzer

Work in progress!

This project is a full-stack LLM application built to help legal professionals quickly review and act on new cases. Users can upload a legal complaint, and a LLaMA-based model will generate a clear summary along with suggested next steps and/or discovery questions for the attorney. The app will also query the CourtListener API (a free legal research platform providing access to millions of court opinions and filings) in real time to identify related cases, helping connect the complaint to relevant precedent. The front end is built in React, with Python handling the LLM and API integrations. Right now the UI components are scaffolded and the backend structure is in place, with the goal of evolving it into a fully production-ready legal analysis tool.
